# Linke Genealogical Engineering

Welcome to **Linke Genealogical Engineering**, a small, independent business combining professional genealogical research with practical software development for genealogists.

This GitHub organization hosts the software side of the business. We focus on creating tools that make historical and genealogical work more structured, transparent, and sustainable.

- **Public repositories** contain small, focused libraries that primarily support data handling. They are published as **NuGet packages** to make them easy to integrate into other projects.
- **Private repositories** hold larger or early-stage applications intended to support genealogical research and documentation processes in the future.

To learn more about what we are currently building and how we approach software development, visit our [Software Services](https://www.linke.engineering/services/software) page.
